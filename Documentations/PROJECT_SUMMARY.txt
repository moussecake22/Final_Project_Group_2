ğŸ› TROUBLESHOOTING COMMON ISSUES:

ISSUE: Ansible can't connect to VMs
FIX: 
  ssh-keygen -t rsa
  ssh-copy-id user@vm-ip
  ansible all -m ping

ISSUE: Docker build fails
FIX:
  Check disk space: df -h
  Verify Docker installed: docker --version
  Check logs: ansible-playbook build-image.yml -vvv

ISSUE: Container not accessible
FIX:
  Check firewall: ansible all -a "firewall-cmd --list-all"
  Check logs: ansible all -a "docker logs portfolio-web"
  Test locally: curl http://localhost:8080

ISSUE: Port 8080 already in use
FIX:
  Find process: netstat -tlnp | grep 8080
  Use different port or stop conflicting service
