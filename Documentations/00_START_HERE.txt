üêõ TROUBLESHOOTING:

CAN'T ACCESS GITHUB?
  ‚Üí Ask repository owner to add you as collaborator
  ‚Üí Check: https://github.com/moussecake22/Final_Project_Group_2/settings/access

ANSIBLE CONNECTION FAILS?
  ‚Üí ssh-copy-id user@vm-ip
  ‚Üí ansible all -m ping -vvv (verbose output)
  ‚Üí Check SSH keys: ssh-add -l

DOCKER BUILD FAILS?
  ‚Üí Check disk space: df -h
  ‚Üí Verify Docker installed: docker --version
  ‚Üí Use verbose: docker build -t team-portfolio . --progress=plain

CONTAINER NOT ACCESSIBLE?
  ‚Üí Check firewall: firewall-cmd --list-all (CentOS)
  ‚Üí Check firewall: ufw status (Ubuntu)
  ‚Üí Check logs: docker logs portfolio-web
  ‚Üí Test locally: curl http://localhost:8080

WEBSITE DOESN'T LOAD?
  ‚Üí Verify content.json is valid JSON
  ‚Üí Check browser console for errors
  ‚Üí Test with: python3 -m json.tool content.json
